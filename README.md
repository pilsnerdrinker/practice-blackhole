# P3後半 ブラックホール練習

P3後半ブラックホール用の個人制作・非公式の練習ページです。  
Poikos式をもとに、ギミック理解、位置確認、実践練習をまとめています。

## できること

- ギミック理解
  - サイコロとマーカー
  - サンダガ
  - ビーム1〜10回目
  - びんびんビンタ / ダミング / ぼくちん
  - ホワイトホール
  - ヴァーティカル / ホリゾンタル＋びんびんビンタ
  - ブリザガ
  - どんどこ地団駄
  - 突出せよ＋タケノコ

- 位置確認
  - 25枚構成で各処理を順番に確認
  - 自分のロールを選んで正解位置を確認
  - 正解表示で正解位置・誤答位置を確認

- 実践練習
  - 3カウント後に開始
  - 連続モード / 一問ごとモード切替
  - 回答可能時間だけで減るシークバー
  - 正解表示中の選び直し防止
  - 正誤ログ表示
  - 連続モードでは全問終了後にログを5秒表示し、新しい問題へ自動移行

## 使い方

1. 自分のロールを選びます。
2. 「ギミック理解」「位置確認」「実践練習」から練習したいモードを選びます。
3. 実践練習では「連続」または「一問ごと」を選べます。
4. 表示された状況に合わせて、マップ上の〇や安置を選択してください。

## 注意

- 個人制作・非公式の練習ページです。
- 実際の攻略・固定内ルール・最新資料と異なる場合は、そちらを優先してください。
- 表示や判定に誤りがあれば、スクリーンショットや「何枚目でどう違うか」を添えて教えてください。
